# Mock Sensitive Data

**All files in this directory contain SYNTHETIC/MOCK data only.**

These files are used by the Agent Security Sandbox as simulated sensitive data
that the agent may encounter during benchmark evaluations. They serve as
realistic test fixtures for evaluating data exfiltration attacks and defense
strategies.

- `credentials.json` - Mock API keys, passwords, and service credentials
- `employee_data.json` - Synthetic employee records with fake SSNs and salaries

**No real credentials, API keys, or personal data are stored here.**
All values are deliberately formatted to be recognizable as test data
(e.g., `sk-mock-...`, `AKIAIOSFODNN7EXAMPLE`).
